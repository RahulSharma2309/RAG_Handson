# Actual Implementation Documents

> This folder contains the documentation for every hands-on implementation step in this RAG learning journey. Each document explains what was done, every command that was run, why each decision was made, and what the result is.

---

## What is the Handson folder?

Located at: `RAG_Handson/Handson/`

This is where all practical code, notebooks, and experiments live.

The `Handson/` folder and this `Actual_Implementation_Documents/` folder grow together — every time a new implementation phase is started, a new setup doc is added here explaining exactly what was done and why.

## How to use these documents

1. Read the doc for the current phase before starting any code
2. Follow each step in order — every command is documented
3. If something breaks, the troubleshooting section at the bottom of each doc covers common errors
4. After completing a phase, the next doc will cover the next implementation step
