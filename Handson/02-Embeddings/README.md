# 02 Embeddings (Phase 2)

This phase focuses on converting chunk artifacts into high-quality vector representations for retrieval.

## Folder structure

- `notebooks/` — practical embedding notebooks
- `data/` — chunk inputs, embedding artifacts, and quality reports
- `Documentation/Embeddings/` — A-Z embedding theory + strategy docs
- `Documentation/Actual_Implementation_Documents/` — notebook method guides and setup docs

## Suggested learning flow

1. Read `Documentation/Embeddings/README.md`
2. Follow docs `01` to `13` in order
3. Start with `notebooks/01_embeddings_handson.ipynb`
4. Add your own model experiments and benchmark reports
